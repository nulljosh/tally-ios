# Tally iOS

iOS companion app for [Tally](https://github.com/nulljosh/tally), the BC benefits tracker.

## Features (Planned)
- Dashboard with payment info and countdown
- Monthly report status and submission
- Push notifications for payment dates

## Stack
- SwiftUI, iOS 17+
- @Observable state management
- BC Self-Serve API (shared backend with web)

## Build
Open in Xcode 15+ and run on simulator or device.
